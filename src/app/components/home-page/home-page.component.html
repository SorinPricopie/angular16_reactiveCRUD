Welcome to my Reactive CRUD Angular 16 project!

What can you find here?

  1. routing with lazy loading, CanActivate & CanDeactivate guards;
  2. angular-in-memory-web-api to mock backend;
  3. rxjs for reactive CRUD operations with async pipe and minimal subscribes;
  4. vanila scss for styling;
  5. reactive forms with default & custom validations;

For sure, there is a lot of room for improvement, but this project is mainly designed for reactive CRUD operations.

Thank you, happy coding!
